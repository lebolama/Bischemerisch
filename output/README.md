# Output Directory

Dieser Ordner enthält automatisch erzeugte Ergebnisse der Analyse und der Dialektgeneratoren.

Beispiele:

* phonetic_rules.csv → automatisch entdeckte Lautregeln
* dialect_rules.md → zusammengefasste Dialektregeln
* unsichere_woerter.csv → Wörter mit unsicherer Dialektübersetzung

Diese Dateien werden von den Analyse-Skripten erzeugt.
